## ПРАВИЛА ГЕНЕРАЦИИ

При создании новых content seeds Claude должен:

1. **Проверить tone:**
   - [ ] Звучит как мои реальные посты?
   - [ ] Нет ли GPT-измов из раздела "Что избегать"?
   - [ ] Используются ли мои характерные приёмы?

2. **Проверить структуру:**
   - [ ] Hook как в моих постах?
   - [ ] Развитие мысли в моём стиле?
   - [ ] Завершение типично для меня?

3. **Проверить специфику:**
   - [ ] Есть конкретные детали (как в моих примерах)?
   - [ ] Есть личная перспектива?
   - [ ] Правильные эмоджи (если я их использую)?

---

**Инструкция по заполнению:**

1. Найдите ваши топ-посты (по engagement или личным предпочтениям)
2. Скопируйте их полностью в секцию "Примеры постов"
3. Для каждого поста напишите почему он сильный
4. Разберите структуру и tone-приёмы
5. Выведите общие паттерны
6. Добавьте примеры того, чего избегать (слабые посты, GPT-измы)
7. Регулярно обновляйте файл новыми удачными постами

### Do's
- Write like you talk to a friend
- Use personal stories and examples
- Be specific, not generic
- Show vulnerability and real experiences

### Don'ts
- Don't use corporate jargon
- Don't write like a textbook
- Don't use AI cliches ("In today's fast-paced world...")
- Don't be preachy or condescending

### Anti-AI Filter
Avoid these patterns that scream "written by AI":
- "Let's dive in" / "Let's explore"
- "In today's world" / "In the age of"
- "Game-changer" / "Revolutionary"
- "It's worth noting that"
- Starting every paragraph with the same structure
- Overly balanced "on one hand... on the other hand"
- Lists where every item has the exact same length

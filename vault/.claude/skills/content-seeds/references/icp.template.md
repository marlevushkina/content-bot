## Как использовать ICP в контент-стратегии

### При создании content seeds:
1. **Проверка релевантности:**
   - Эта тема важна для моей ЦА?
   - Решает ли это их реальную боль?
   - Говорю ли я на их языке?

2. **Адаптация tone:**
   - Использую ли термины которые они понимают?
   - Уровень детализации подходит для их экспертизы?
   - Tone соответствует их ожиданиям?

3. **Выбор формата:**
   - Формат который они предпочитают?
   - Длина поста оптимальна?
   - Платформа где они активны?

### Вопросы перед публикацией:
- [ ] Моя ЦА узнает себя в описанной проблеме?
- [ ] Я говорю на их языке (не слишком просто/сложно)?
- [ ] Контент решает их реальную боль?
- [ ] Hook зацепит их в первых 2 строках?
- [ ] Формат подходит для их потребления?

---

**Инструкция по заполнению:**

1. **Проанализируйте вашу текущую аудиторию:**
   - Кто комментирует ваши посты?
   - Кто пишет в личку?
   - Кто становится клиентами?

2. **Изучите аналитику:**
   - Демография подписчиков (LinkedIn, Instagram, и т.д.)
   - Какие посты получают лучший отклик?
   - Откуда приходят люди?

3. **Проведите интервью:**
   - Поговорите с 5-10 представителями вашей ЦА
   - Узнайте их боли своими словами
   - Спросите где они ищут информацию

4. **Регулярно обновляйте:**
   - ICP меняется по мере роста аудитории
   - Добавляйте новые инсайты из общения
   - Тестируйте гипотезы и уточняйте профиль
